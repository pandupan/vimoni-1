# 🏗️ Vimoni

Aplikasi berbasis web dengan menggunakan Next.js. Proyek ini dikembangkan untuk menyediakan layanan atau platform digital tertentu.

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
